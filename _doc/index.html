The documentation is created and generated using ciigo.

First, install ciigo command line interface,

$ go install git.sr.ht/~shulhan/ciigo/cmd/ciigo

and then run

$ ciigo serve _doc/
## or
$ make docs-serve

to serve documentation on HTTP server at http://127.0.0.1:8080 and watch changes on documentation files (".adoc") and regenerate the HTML files.

To generate HTML files for all asciidoc files, run

$ ciigo convert _doc/
## or
$ make docs

That’s it! Happy hacking!
